<h1 align="center">
    Instruções
</h1>

Basta clonar este repositório.

<br>
Instalar as dependências, utilizando o comando:
<br>
<pre><code>npm install</code></pre>
<br>
Rodar o projeto, utilizando o comando:
<pre><code>node .</code></pre>
<br>
Digitar um número inteiro positivo:
<br>
